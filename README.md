## Tech Stack Used

```

1. Bootstrap 4
2. NodeJS(EJS, Express & related modules)
3. HTML 5
4. CSS

```

## Run the app 

```

1. Install npm, express, body-parser, ejs & related packages
2. Open terminal ➡️ node app.js
3. Open localhost:3000 on your browser
4. Enjoy the app 🎉
5. Star the repo 😎

```
